<h1>Spring Cloud Config Server using Spring Boot </h1>

## Steps

- Clone as follows 

```
git clone https://github.com/derrick81/scs-config-master.git
```

- Package as shown below

```
$ ./mvnw -DskipTests package
``` 

- Run as follows

```
$ ./mvnw spring-boot:run
```
